# copycat
Test data runner


### Setup 

Use python 3.9
``` 
pip install -r requirements.txt
```

### Run

``` 
flask run
``` 

Endpoint: 
```
http://127.0.0.1:5000/validate/org_id/{{org_id}}/design/{{design_name}}
```
